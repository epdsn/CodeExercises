## How to Use This List

1. **Daily Practice**: Pick 1-3 problems per day based on your available time
2. **Topic Focus**: Spend a week on each category to build depth
3. **Difficulty Progression**: Start with arrays/strings, progress to DP and advanced topics
4. **Review**: Revisit problems after 1 week, 1 month, and 3 months
5. **Optimization**: Solve each problem, then optimize for time and space complexity
6. **Testing**: Write unit tests for each solution

## Tips for Success

- Always analyze time and space complexity
- Consider edge cases: empty arrays, null values, single elements
- Practice explaining your solution out loud
- Use LINQPad or .NET Fiddle for quick testing
- Compare your solution with others after solving
- Focus on clean, readable C# code with proper naming conventions

## Getting Started

### Prerequisites
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) or later

### Running the Application

1. **Clone or navigate to the project directory**
   ```bash
   cd c:\Users\ep1\source\repos\CodeExercises
   ```

2. **Restore dependencies**
   ```bash
   dotnet restore
   ```

3. **Build the project**
   ```bash
   dotnet build
   ```

4. **Run the application**
   ```bash
   dotnet run
   ```

   **Important:** Always use `dotnet run` to run the entire project, not `dotnet .\Program.cs`

5. **Follow the prompts**
   - Type `y` when asked if you're ready to start
   - The program will run the current active exercise

### Project Structure

```
CodeExercises/
├── Program.cs                          # Main entry point
├── Exercises/                          # Your solution implementations
│   ├── TwoSums.cs                     # Example: Two Sum problem
│   └── [Add more problems here]
├── CodeExcerciseDataImportOutput.csproj # Project file
└── README.md                           # This file
```

### Adding New Exercises

1. Create a new `.cs` file in the `Exercises` folder
2. Use the `Exercises` namespace
3. Implement your solution class
4. Update `Program.cs` to run your new exercise

**Example template:**
```csharp

            // Your implementation here
        }
    }
}
```
